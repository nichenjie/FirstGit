# 三栏策略界面设计

## 概述

重新设计金融数据页面布局，将原有的"二档策略+单页百分位图"改为"三栏独立策略栏目"，每栏可独立选择统计周期和阈值参数，新增 3 个月和 6 个月短线周期，移除二档设置功能。

## 布局结构


## 页面区块说明

### 1. 顶部价格走势图（不变）

- **内容**：新城控股（CNY左轴）+ 新城发展（HKD右轴）双Y轴对比图
- **数据范围**：始终显示全部历史数据，**不随**下方策略栏周期筛选变化
- **位置**：占据页面主体区域上半部分，横向拉满

### 2. 三栏策略区（核心变更）

每栏结构一致，独立运行：

#### 控制区
- **统计周期下拉框**：可选 3个月、6个月、1年、2年、3年、5年
- **低阈值输入框**：低于 X% 时买入
- **高阈值输入框**：高于 X% 时卖出
- **智能策略分析按钮**：独立分析本栏参数

#### 百分位走势图
- **只保留一条金色百分位折线**（移除所有一档/二档额外标记线）
- **买入线**：绿色虚线（对应低阈值）
- **卖出线**：红色虚线（对应高阈值）
- **独立 ECharts 实例**，每栏独立刷新

#### 信号与收益区
- 信号按钮组（点击联动顶部价格图滚动到对应日期）
- 交易区间收益分析
- 当前持仓状态（距下一次交易的百分位距离）

### 3. 数据详情表格（保留，位置调整）

- 移至三栏策略区下方
- 始终显示全部历史数据的明细，不受栏目筛选影响
- 列：日期、收盘价、市值、汇率、人民币市值、市值比、百分位

## 交互行为

| 操作 | 行为 |
|------|------|
| 修改任意栏的周期/阈值 | 该栏**实时自动**重新计算并刷新图表和信号 |
| 点击智能策略分析 | 本栏弹出分析结果（最优阈值），可一键应用 |
| 点击信号按钮 | 顶部价格图联动滚动到对应日期 |
| 刷新数据 | 全量刷新，所有栏同步更新 |

## 移除内容

- ❌ 二档设置（tier2 config）整个区块
- ❌ 百分位图上的一档/二档额外阈值标记线（markLine）
- ❌ `tier2-low-threshold`、`tier2-high-threshold`、`tier2-position-ratio` 等UI控件
- ❌ 右侧 signal-panel 侧边栏布局（改为三栏平铺）

## 文件变更范围

| 文件 | 变更内容 |
|------|----------|
| `index.html` | 重构布局：移除 signal-panel 侧边栏，将三栏百分位图和控制区嵌入主区域；新增3个月/6个月选项 |
| `charts.js` | 每个策略栏独立初始化一个 ECharts 实例；移除 `updateTieredChart`/`updateSignalMarkers` tier2 相关逻辑 |
| `main.js` | 每个策略栏独立的事件监听+计算逻辑；实时自动刷新；移除 tier2 相关代码和双档findSignalsForTier调用 |
| `style.css` | 新增三栏布局样式（`.strategy-columns`）；调整 `.chart-container` 为独立栏目样式；响应式 breakpoint |
